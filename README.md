# Kaif by Fifi - Jewelry E-commerce Website

A beautiful e-commerce website for jewelry, built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Elegant homepage with hero section and featured products
- Product listing page with all available jewelry
- Detailed product pages with images and descriptions
- Responsive design optimized for all devices
- Sample products including rings, necklaces, earrings, and bracelets

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the website

## Project Structure

- `src/app/` - Next.js App Router pages
- `src/components/` - Reusable React components
- `src/data/` - Sample product data
- `public/` - Static assets

## Technologies Used

- Next.js 16
- TypeScript
- Tailwind CSS
- ESLint
